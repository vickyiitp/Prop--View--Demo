# Prop--View--Demo 🏠

Prop--View--Demo is a high-end Real Estate Property Viewing application designed to provide clients with a premium, interactive experience for exploring properties. Built with **React 19**, **Vite**, and **Tailwind CSS**, it combines performance with a stunning visual design powered by **Framer Motion**.

---

## ✨ Key Features

- **🏢 Interactive Property Showcase**: Explore properties with high-quality images, detailed descriptions, and contextual metadata.
- **📊 Real-Time EMI Calculator**: Integrated mortgage calculator allowing clients to estimate monthly payments instantly based on property price, down payment, and interest rates.
- **🔍 Smart Property Filtering**: Advanced search and filter capabilities to narrow down properties by type, location, and price range.
- **📱 Responsive & Premium UI**: A fully responsive design optimized for desktop and mobile, featuring smooth micro-animations and a glassmorphic aesthetic.
- **🤖 AI-Powered Property Insights**: (Experimental) Integration with **Google Generative AI** (Gemini) for smart property descriptions and data analysis.

---

## 🛠 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **AI Integration**: [Google Generative AI](https://ai.google.dev/) (Gemini API)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/vickyiitp/Prop--View--Demo.git
   cd Prop--View--Demo
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file in the root directory and add your API keys:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Launch the Development Server**:
   ```bash
   npm run dev
   ```
   *The application will be available at `http://localhost:3000`.*
